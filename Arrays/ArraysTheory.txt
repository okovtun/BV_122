Массив (Array) - это набор переменных одного типа в непрерывной области памяти.
1. Динамические и Статические;
2. Как статические, так и динамические массивы бывают:
   одномерные, двумерные, трехмерные, и т.д.... многомерные.


						ОБЪЯВЛЕНИЕ МАССИВОВ:

				type name;		//Объявление переменной
				type name[SIZE];//Объявление массива
type - тип элементов массива. Элементы массива могут быть любого существующего типа.
	   Элементы одного определенного массива могут быть только одного определенного типа.
	   В одном массиве НЕ может быть один элемент int, другой double, третий char.
name - имя массива. Для именования массивов используются такие же идентификаторы, 
	   как и для именования переменных.
SIZE - количество элементов (размер) массива. Может быть задано лишь целочисленным конствнтным значением, 
	   и только во время написания кода.
